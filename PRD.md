# PRD: 政府標案情資查詢系統

版本：v1.0  
更新日期：2026-07-06  
狀態：MVP 已完成，進入部署與後續擴充階段

## 1. 產品目標

建立一個可查詢政府標案資料的 Streamlit 網頁工具。使用者輸入關鍵字或簡單自然語言後，可以跨年度搜尋標案，快速查看機關、標案名稱、得標廠商、協作廠商、金額、年度分布與摘要統計，並匯出查詢結果。

本系統優先服務「政府標案情資初步盤點」情境，協助使用者從既有 CSV / SQLite 資料中快速找出特定機關、得標廠商或 SI 相關標案，而不是取代正式採購資料庫或 BI 系統。

## 2. 使用者與使用情境

主要使用者：

- 業務、售前、產業研究或標案分析人員。
- 需要快速盤點特定機關歷年資訊服務採購紀錄的人員。
- 需要比對得標廠商、協作廠商與金額分布的人員。

核心情境：

1. 使用者輸入「中華郵政 SI」或「查中華郵政近五年最大 SI」。
2. 系統解析關鍵字、年度範圍與排序條件。
3. 系統顯示查詢摘要、年度趨勢、機關排行、得標廠商排行與明細。
4. 使用者下載查詢結果 CSV 或摘要報表。

## 3. 資料範圍

目前資料來源為：

```text
資料庫_CSV/*SourceData.csv
```

資料會透過 `scripts/import_to_sqlite.py` 匯入：

```text
database/tenders.sqlite
```

目前已知資料涵蓋 2021-2025 年 CSV。若未來取得 2020 或 2026 資料，應沿用相同匯入流程納入 SQLite。

主要欄位對照：

| 顯示名稱 | CSV 欄位 / 來源 |
|---|---|
| 年度 | 從檔名 `award_YYYY...` 解析 |
| 機關 | `TenderOrgName` |
| 標案名稱 | `TenderName` |
| 金額 | `TenderAwardPrice` / `TenderAwardPriceNumber` |
| 得標廠商 | `BidderSuppName` |
| 協作廠商 | `NotObtainSuppName` |
| 標案案號 | `TenderCaseNo` |
| 決標日期 | `AwardDate` |
| 採購類型 | `ProcurementType` |
| 採購屬性 | `ProcurementAttr` |
| 決標方式 | `TenderAwardWay` |

完整欄位說明請見 `docs/FIELD_MAPPING.md`。

## 4. MVP 功能需求
### 4.0 登入

- app 進入查詢頁前需顯示登入頁。
- 首頁提供申請帳號機制，新申請帳號需等待管理員審核。
- 使用者需輸入帳號與密碼才能查看資料。
- 帳密優先讀取環境變數或 Streamlit secrets；未設定時使用本機預設帳密。
- Streamlit secrets 可設定後台核准帳號清單，只有 `approved = true` 的帳號可以登入。
- 管理員登入後可進入後台管制介面，核准、停用帳號與調整管理員權限。
- 登入嘗試需留下審核紀錄，紀錄時間、帳號與狀態，不記錄密碼。
- 登入狀態保存在 Streamlit session state，並在頁首右上角提供原生登出按鈕，位置靠近 Share 區域。

### 4.1 資料載入

- 從 SQLite 讀取標案資料。
- 找不到 SQLite 時，允許退回讀取 CSV，並提示使用者應先執行匯入腳本。
- 載入結果使用 Streamlit 快取，避免每次互動都重新讀取資料。
- 資料缺欄、資料庫缺表或資料為空時，顯示友善錯誤訊息。

### 4.2 查詢

- 支援關鍵字搜尋。
- 搜尋欄位包含機關、標案名稱、得標廠商、協作廠商。
- 支援跨年度查詢。
- 支援年度篩選。
- 支援簡單自然語言解析，例如近三年、近五年、最大金額、最小金額。

### 4.3 結果呈現

- 顯示查詢結果表格。
- 表格欄位包含年度、機關、標案名稱、金額、得標廠商、協作廠商、案號、日期與來源資訊。
- 金額以千分位格式顯示。
- 預設依年度新到舊排序，並支援金額高到低、金額低到高排序。
- 查無資料時顯示「查無相關標案」。
- 查詢結果過多時，畫面分頁顯示，下載仍包含完整結果。

### 4.4 摘要與分析

- 顯示本次查詢筆數。
- 顯示總金額。
- 顯示年度分布。
- 顯示前 3 機關與前 3 得標廠商。
- 顯示年度趨勢圖。
- 顯示機關排行與得標廠商排行。
- 支援單一機關分析。
- 支援單一得標廠商分析。
- 支援單筆標案明細。

### 4.5 匯出

- 查詢結果 CSV 下載。
- 摘要報表 CSV 下載。
- 固定格式機關分析報告 CSV 下載。
- `scripts/query_agency_si.ps1` 可查詢指定機關 SI 金額並輸出 CSV。

### 4.6 規則式智慧功能

- 自然語言查詢解析。
- 規則式文字洞察。
- 相似標案推薦。
- 異常提醒。

目前智慧功能不串接真正 LLM，僅以規則與資料統計產生摘要。

## 5. 非功能需求

- Windows PowerShell 可啟動。
- 本機啟動腳本為 `run_app.ps1`。
- Python 依賴列於 `requirements.txt`。
- 修改 Python 後至少執行語法檢查與測試。
- CSV 原始資料不可被 app 執行流程修改。
- SQLite 匯入流程需可重跑。
- 大量資料效能優先透過 SQLite 與資料匯入流程處理，避免在 Streamlit 內堆疊過多複雜 pandas 邏輯。

## 6. 非目標

v1.0 不包含：

- 多角色權限控管。
- 正式外部 API 自動同步。
- 真正 LLM 問答。
- 即時採購公告爬取。
- 完整 BI 儀表板。
- 多使用者資料權限隔離。

## 7. 驗收標準

1. 使用者可在本機啟動 Streamlit app，並先看到登入頁。
2. app 可讀取 SQLite 或退回 CSV。
3. 使用者可輸入關鍵字並取得跨年度查詢結果。
4. 查詢結果欄位可讀，金額格式正確。
5. 查無資料時有明確提示。
6. 使用者可依年度與金額排序。
7. 使用者可篩選年度。
8. 使用者可下載查詢結果 CSV。
9. 使用者可查看摘要、趨勢、排行、單一機關、單一得標廠商與標案明細。
10. Python 語法檢查與 pytest 通過。
