## 目前功能

1. 啟動 Streamlit 網頁。
2. 快取載入 CSV 資料，避免每次互動都重讀檔案。
3. 輸入關鍵字後，從所有年度資料查詢包含關鍵字的標案。
4. 搜尋欄位包含機關、標案名稱、得標廠商、協作廠商。
5. 顯示查詢摘要與查詢結果表格。
6. 支援年度新到舊、金額高到低、金額低到高排序。
7. 支援年度篩選。
8. 支援查詢結果 CSV 下載。
9. 找不到 SQLite 資料庫、資料表缺漏或資料為空時，顯示友善錯誤訊息。
10. 查詢結果超過 200 筆時，表格改為分頁顯示，下載仍包含完整結果。
11. 使用 `scripts/import_to_sqlite.py` / `scripts/import_to_sqlite.ps1` 可重建 SQLite 資料庫。
12. 顯示年度趨勢圖、機關排行與得標廠商排行。
13. 提供單一機關與單一得標廠商分析分頁。
14. 提供單筆標案明細分頁，顯示案號、日期、採購類型、決標方式與來源。
15. 提供摘要報表 CSV 下載。
16. 提供固定格式機關分析報告 CSV 下載。
17. 已建立版本節點規劃文件 `docs/VERSION_PLAN.md`。
18. 已建立小型測試資料集與 pytest 測試。
19. 已建立 `.gitignore` 並清理快取與 log 檔。
20. 已建立部署規劃文件 `docs/DEPLOYMENT.md`。
21. 支援自然語言查詢解析，例如近三年/近五年與最大/最小金額。
22. 提供規則式文字洞察、相似標案推薦與異常提醒。
23. 已建立外部 API 串接評估文件 `docs/API_EVALUATION.md`。

## 目前限制

- 資料來源只涵蓋目前存在的 2021-2025 CSV 檔；PRD 中疑似提到 2020-2026，但資料尚未齊全。
- 名稱標準化目前只處理括號與空白差異，尚未建立別名對照表。
- 尚未實作登入、權限控管與正式雲端資料更新流程。
- 文字洞察目前為規則式摘要，尚未串接真正 LLM。
- 外部 API 仍停留在評估階段，尚未納入正式資料匯入。
