# 版本節點規劃

更新日期：2026-06-30

## MVP

目標：完成可穩定查詢的本機 Streamlit 工具。

範圍：

- SQLite 匯入與查詢。
- 跨年度關鍵字搜尋。
- 年度篩選、排序、分頁。
- 查詢摘要與明細表格。
- 查詢結果 CSV 下載。
- 基礎趨勢、排行、單一機關與單一得標廠商分析。
- 單筆標案明細。
- 摘要報表與固定格式機關分析報告匯出。

完成判準：P0、P1、P2、P3 完成，並通過 `py_compile app.py`。

## v1.1：品質與維護

目標：讓功能可重複驗證，降低修改風險。

範圍：

- 建立小型測試資料集。
- 加入資料載入、搜尋、排序、金額格式化與摘要統計單元測試。
- 加入 Streamlit smoke test。
- 建立 `.gitignore` 並清理暫存、log、pycache。
- 補充資料品質檢查。

## v1.2：報表與分析強化

目標：讓使用者更快產出可交付分析。

範圍：

- 增加金額區間、機關、廠商等進階篩選。
- 增加廠商名稱別名對照表。
- 擴充機關分析報告格式。
- 規劃部署方式：本機、內網主機或雲端。

## v2.0：AI 與外部資料

目標：把查詢工具升級為情資分析工具。

範圍：

- 自然語言查詢。
- AI 摘要與洞察。
- 相似標案推薦。
- 異常值提醒。
- 評估串接政府採購公開資料 API 或 g0v/PCC 相關資料源。
